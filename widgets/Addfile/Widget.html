This tool allows you to add a zipped shapefile or Comma Separated Value (.csv) table to the map. For best performance, please limit the number of features to less than 1,000. This is not a hard cutoff, but most modern browsers will start to experience instability with increasing numbers of features. CSV tables should contain fields with Latitude and Longitude coordinates in decimal degrees.
