Tabs provide three different ways to add data to the EnviroAtlas map:

  1. Search – search for data layers in EPA's GeoPlatform, the Federal GeoPlatform, or ArcGIS Online [Exit EPA] and add the layers to the EnviroAtlas interactive map.
  2. URL – add dynamic map services (e.g. image, feature, or tiled services) as well as Open Source WMS and KML services to the EnviroAtlas interactive map from a known URL.
  3. File – upload zipped shapefiles, CSV (comma-separated variables), GeoJSON, KML, or GPX files from a local drive to the EnviroAtlas interactive map.

Search

Use the dropdown to select the data source. Browse the list of layers. You may use the Search box to narrow the list by entering a search term or phrase.

URL

Use the Type dropdown to select the type of data referenced in your URL.

Enter the URL and Layer Name (if appropriate). Click the ADD button.

If the service contains multiple layers, you will see all the layers in the Layer List.

File

Click the question mark icon to view information about data layer uploads. Click Browse to locate the layer file on a local drive.

Once added, you may view the layer and its legend in the Layer List.
