Compare My Area tool

The Compare My Area tool allows users to obtain a report that details characteristics about a location selected in the EnviroAtlas interactive map.

  • Review the text in the tool to select the desired Report type: Watershed or Census Tract.
  • Select the location in the map that you would like to generate the report for. Enter an address or place in the text box and press the magnifying glass icon or interactively select a location in the map with the mouse cursor.
  • The HUC12 subwatershed or Census Tract will display in the map with a pop-up. Click Watershed report or Tract report in the pop-up to open the report in a new browser tab.
