Select Report type

Select the desired Report type: Watershed or Census Tract.

Provide location of interest

Select the location of interest using one of these methods:

  1. Interactively select the location in the interactive map using the mouse cursor.
  2. Enter the address or place name in the text box. A dropdown list of potential locations are displayed. Select the desired location from the list.

The EnviroAtlas interactive map zooms to the desired location, displaying the subwatershed or census tract for the location selected.

Open Report

In addition to displaying the subwatershed or census tract, a pop-up appears. The pop-up contains the HUC12 name and number or the census tract FIPS code and an option to open the report by clicking Watershed report or Tract report.

The report opens in a new browser tab. Watershed reports provide a map of the subwatershed and county along with graphs and tables comparing Landscape Indicators between the selected subwatershed, the primary county in which the subwatershed resides, and the primary state in which the subwatershed resides.

Census tract reports provide three tabs containing graphs and tables comparing attributes of the selected tract, the primary county in which the tract resides, and the primary state in which the tract resides. Click the tab to reveal the data:

  • Demographic Indicators – demographics from the American Community Survey.
  • National Air Toxics Assessment – air toxics data from NATA.
  • Data Table – consolidated table of demographics and NATA data.
