Draw and Measure

Use Draw and Measure to add custom markup to the map. Add points, lines, areas, or text to the map for reference or to obtain map coordinates, measure distances, and measure areas.

  • Select the drawing mode (graphic element or text). The Point element returns geographic coordinates, Line elements measure distance, and Area (polygon) elements measure area. The Text element adds text to the map.
  • Style the drawing as desired (e.g. colors, line widths, point style) and name the drawing.
  • Interactively add the graphic element or text to your map.
