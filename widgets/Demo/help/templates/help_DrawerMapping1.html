Mapping Tools

This toolbox contains four Mapping Tools:

Enhanced Bookmarks – use spatial bookmarks to quickly zoom the interactive map directly to the conterminous USA or to an EnviroAtlas community, or create your own spatial bookmarks that can be saved and reused.

Print – export a graphic file of the current map view in a variety of file formats.

Draw and Measure – add points, lines, areas, or text to the map for reference or to obtain map coordinates, measure distances, and measure areas.

Legend – view a pop-up map legend separate from the legend in the Layer List window.
