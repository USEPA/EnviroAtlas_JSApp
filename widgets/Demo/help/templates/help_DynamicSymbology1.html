Dynamic Symbology

Use the Dynamic Symbology tool to modify the appearance of the map layer. You may modify colors, symbols, classification scheme, and transparency of the layer in the interactive map.

By default, most EnviroAtlas layers display in the map with a seven-class Geometric Interval classification scheme symbolized with a yellow-green-blue color ramp at approximately 45% transparency.

  • Change layer colors using the Symbology dropdown.
  • Change the classification scheme using the Classification dropdown.
  • Adjust the number of classes using the # of Classes dropdown.
  • Adjust the transparency of the layer in the map using the slider bar.
  • Apply changes with the Apply button. Go back to the default with the Reset button.
