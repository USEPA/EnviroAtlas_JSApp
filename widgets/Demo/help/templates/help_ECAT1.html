EnviroAtlas Change Analysis Tool (ECAT)

The EnviroAtlas Change Analysis Tool allows a selected climate variable to be compared between time periods. The calculations produce a layer in the interactive map that depicts the change in the variable between the two time periods. The tool includes historic data as well as modeled future scenarios.

  • Take note of the recommendation text in the tool pop-up.
  • Select the parameters for the comparison from the seven dropdown menus.
  • Click Calculate Change to add the ECAT layer to the map.
  • Click Clear Layers to remove the layer from the map.
  • Click Download HUC 12 average values (CSV) to download a comma-separated-value file of the average change values by HUC12.
