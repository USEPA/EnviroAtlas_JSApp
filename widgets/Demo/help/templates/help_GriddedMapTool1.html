Gridded Map Tool

The Gridded Map Tool allows users to summarize EnviroAtlas metrics (national indicators) from a set of gridded layers within a user-selected focus location area.

  • Select a national indicator gridded layer from the dropdown list.
  • Select a unit area from the dropdown list.
  • Select, create, or upload the focus location area in the map.
  • Click the Calculate button.
  • View the the gridded layer statistics for the focus location in the Results tab.
