Gridded Map Tool

The Gridded Map Tool allows users to view a gridded layer in the EnviroAtlas interactive map and extract statistics from the layer within a user-selected unit area.

  • Select a unit area from the dropdown list.
  • Click the desired location in the map.
  • Select the gridded layer from the dropdown list.
  • Click the Calculate button.
  • View the the gridded layer statistics for the selected area in the Results tab.
