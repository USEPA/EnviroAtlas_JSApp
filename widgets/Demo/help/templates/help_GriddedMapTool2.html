In the Gridded Map Tool, select a national indicator from the dropdown:

Select the unit area from the dropdown:

  • County – select a county.
  • State – select a state.
  • Congressional District – select a Congressional District.
  • HUC-8 – select a HUC8 subwatershed. Note: zoom in to view HUC boundaries.
  • HUC-12 – select a HUC12 subwatershed. Note: zoom in to view HUC boundaries.
  • Draw a point – click to create a point in the map and designate a buffer area around it.
  • Draw a line – draw a line in the map and designate a buffer area around it.
  • Draw an area – draw an area in the map. Optionally designate a buffer area around it.
  • Use your own shapefile – upload your own shapefile.

Select, Create, or Upload the focus location interactively in the map:

  • Select a focus location on the map using the County, State, Congressional District, HUC-8, or HUC-12 options:
    • Click the selected focus location from the layer in the map. Zooming in on the map may be necessary to find the appropriate location. Only one focus location may be selected.
  • Create a focus location area on the map using the Draw a point, Draw a line, or Draw an area options:
    • To Draw a point, first set the buffer measurement units (km or mi) and buffer distance in the tool. Then in the map, click the mouse button at the desired focus location. The focus location area will appear in the map as a circle, centered on the clicked point, with a radius equal to the buffer distance.
    • To Draw a line, first set the buffer measurement units and buffer distance in the tool. Then in the map, click the mouse button and hold down to draw a freehand line. Let go of the mouse button to finish the line. The focus location will appear as an area on the map following the linear path drawn, buffered by the buffer distance.
    • To Draw an area, optionally set the buffer measurement units and buffer distance in the tool. Check the box to exclude inner features of the area, if desired. Click the mouse button and hold down to draw a freehand focus location area in the map. Let go of the mouse button to finish the drawing. The focus location area will appear, including the buffer if selected.
  • Upload a zipped shapefile to create a focus location area on the map:
    • Click Choose File to navigate to a polygon-type shapefile stored on a local computer.
    • The uploaded geographic shape will appear in the map as the focus location.

In the example below, the National Land Cover Dataset (NLCD) is selected as the national indicator from the dropdown. For this indicator, choices for NLCD Layer Year are presented in a separate dropdown. Year 2016 is selected. County is selected as the unit area, and County boundaries are displayed in the map.

In the map, click a County to designate the focus location to analyze. The EnviroAtlas map zooms to the extent of the County, in this case Sweet Grass, Montana. The Calculate button becomes active.

Click Calculate. The Gridded Map Tool extracts the statistics (Area and Percentage for each 2016 NLCD Land Cover Type) within Sweet Grass County. Statistics appear in a table in the Tool's Results tab. Scroll through or resize the Gridded Map Tool window to view values in the table. Click Print to export a graphic image of the table.
