EnviroAtlas HUC12 Navigator

The EnviroAtlas HUC12 Navigator tool allows the user to click on a location in the interactive map (within the conterminous US) and view HUC (hydrologic unit code) subwatersheds upstream or downstream from the selected location within the hydrologic network.

  • Select the navigation direction: Upstream or Downstream.
  • Click the map location to add the origination point for the navigation.
  • View the upstream or downstream HUCs in the map.
  • View the statistics for the HUCs in the tool's Select Subwatershed tab.
  • View EnviroAtlas metrics aggregated to the selected HUCs in the tool's Aggregate Indicators tab.
