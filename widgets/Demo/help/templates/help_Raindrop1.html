Raindrop Tool

The Raindrop tool generates a flow path and calculates the distance from an interactively selected point on the map to the nearest water feature in the National Hydrography Dataset Plus (NHDPlus). The flow path follows downslope terrain in a digital elevation model (DEM).

  • Note: Display the NHDPlus V2 features layers (located in the Hydrologic Features layer theme) before using this tool to provide visual context about the hydrologic network.
  • Click the Activate Tool button.
  • Click the map location to start the downstream flow path.
  • View the flow path in the map.
  • View the flow path distance in the tool's Results tab.
  • Adjust the performance of the tool in the tool's Settings tab..
