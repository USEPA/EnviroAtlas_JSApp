Saving a session

Prepare the EnviroAtlas interactive map session as desired (i.e. zoom to a location, add layers of interest, etc.). Open the SaveSession tool.

To create your saved session, add a name for the session in the Name: text box.

Click the Save button. The name will appear under Saved Sessions in the list.

Multiple sessions may be saved. The Saved Sessions list contains each of the sessions.

Actions on saved sessions

Hover over the name of a Session to view the options available.

  • Load Map – load the saved session.
  • Download Map – download the saved session to a local drive as a JSON file.
  • Edit – edit the name of the saved session.
  • Move Down – move the saved session down the list.
  • Delete – delete the saved session.

NOTE: Saved sessions are stored in your browser cache and may be permanently deleted if the cache is cleared. Preserve your saved sessions if this is a concern.

Preserving saved sessions

Click Save to file to preserve saved sessions. Save to file compiles the parameters of all of the saved sessions in a JSON file that you may save locally.

Click Load from file to load the JSON file and recover saved sessions.

Saved session JSON files may be shared with other EnviroAtlas interactive map users.
