Select

Use the Select tool to interactively select features from the EnviroAtlas Data layers that have been added to the map. Features may be selected from one or more layers in the map.

  • Make sure desired layers are selectable in the map following the instructions in the tool.
  • Use the Select dropdown to choose the selection method.
  • Interactively draw the selection area on the map. Follow the tooltips provided to start and end the drawing.
  • Click the layer name in the Select tool to view the legend for the selected features.
  • Click the ellipsis to the right of the layer name in the Select tool to view supported actions for the selected features.
  • Click the Clear button in the Select tool to unselect the selected features.
