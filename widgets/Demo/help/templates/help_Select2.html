Interactively select features

The Select tool allows the user to interactively select a subset of 1,000 or fewer features displayed in the map and perform actions on those features.

Use the Select dropdown to choose the type of interactive drawing to use to select the features and draw on the EnviroAtlas interactive map.

The map displays the selected features that intersect with the drawing with light blue outlines in the map. The number of features selected appears next to the layer name in the Select tool.

Perform actions on the selected features

Click the ellipsis to the right of the layer name to view the actions that may be taken on the selected features in the layer (Note: not all actions are available for all layers):

  • Zoom to – zooms the map to the extent of selected features.
  • Pan to – pans the map to the center point of the selected features.
  • Flash – makes the selected features in the map flash.
  • Export to CSV file – exports data for the selected features to a comma-separated value (CSV) file.
  • Export to feature collection – exports data for the selected features to a JSON file.
  • Export to GeoJSON – exports data for the selected features to a GeoJSON file.
  • Statistics... – produces a pop-up with a dropdown menu of fields in the layer's attribute table. Select a field to view statistics for the selected features.
  • Create layer – creates a temporary layer of the selected features in the map.
  • Save to My Content – saves the selected features as a feature layer in a specified folder in the user's EPA GeoPlatform Online account.
  • View in Attribute Table – opens the attribute table for the layer containing only the selected features.
  • Clear selection – clears the selected features.

Clear button

Click the Clear button to unselect the features from the layer.
