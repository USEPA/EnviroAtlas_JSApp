Interactively select features

The Select tool allows the user to interactively select a subset of features displayed in the map and perform actions on those features.

Use the Select dropdown to choose the type of interactive drawing to use to select the features and draw on the EnviroAtlas interactive map.

Displayed map layer features that intersect with the drawing are selected and appear with light blue outlines in the map. The number of features selected appears next to the layer name in the Select tool.

Perform actions on the selected features

Click the ellipsis to the right of the layer name to view the actions that may be taken on the selected features in the layer (Note: not all actions are available for all layers):

  • Zoom to – zooms the map to the extent of selected features.
  • Pan to – pans the map to the center point of the selected features.
  • Flash – makes the selected features in the map flash.
  • Export to CSV file – exports data for the selected features to a comma-separated value (CSV) file.
  • Export to feature collection – makes the selected features in the map flash.
  • Export to GeoJSON – makes the selected features in the map flash.
  • Statistics... – makes the selected features in the map flash.
  • Create layer – makes the selected features in the map flash.
  • Save to My Content – makes the selected features in the map flash.
  • View in Attribute Table – makes the selected features in the map flash.
  • Clear selection – makes the selected features in the map flash.

Clear all topics

Click the clear all topics checkbox in the top bar to restore the full data theme list.
