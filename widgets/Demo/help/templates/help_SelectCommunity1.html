Community Selection

The Community Selection tool focuses the interactive map display on an EnviroAtlas community selected by the user. The tool zooms the map to the community and tailors community-level map layers to represent only the selected community.

  • Select an EnviroAtlas community from the list.
  • The map automatically zooms to the extent of the community and display the community name in the upper left corner of the map.
  • Community data layers display based only on the selected community's data.
  • To return to the default view, select Combined Communities.
