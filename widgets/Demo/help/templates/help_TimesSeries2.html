Video Example [Exit EPA]

Use the dropdown menus to select the desired climate data variables. Click on the circled i to view descriptions of the data choices.

  1. Scenario– select the climate model to map a future scenario, or select “Historic .”
  2. Climate Variable– select the variable to be mapped.
  3. Season– select the season to be displayed on the map, or “Annual” for annual mean values.

Allow time for the map to calculate the values and open the data layer corresponding to the selected entries. Along with the map display, the Layer List opens with the climate layer added. You may view the legend for the layer by clicking on the layer name in the Layer List.

The Time Series Tool includes an animation function to view predicted future or historic map displays. You may play and pause the animation with the Play/Pause button. When paused, you may step forward or backward through time with the Previous and Next buttons.
