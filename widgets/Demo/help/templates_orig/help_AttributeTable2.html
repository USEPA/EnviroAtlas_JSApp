The Attribute Table displays below the interactive map and includes several tools to control the appearance of the table.

  1. Data layer tabs – View the Attribute Table for one of the data layers in the map by selecting the layer's tab.
  2. Table tools – Control the appearance of the Attribute Table with tools in this toolbar.
  3. Resize table – Pull the divider up or down to resize the table.
  4. Show/hide columns – Click the plus button to view a list of columns (fields) and turn them on or off in the table.
  5. Column (field) names – View the column names for the active data layer at the top of the Attribute Table.
  6. Field values – View the values for features in the active data layer.

Data layer tabs

Tabs across the top of the Attribute Table appear for each layer in the map Layer List. Click the tab to view that data layer's table.

Table tools

Table tools allow you to adjust the appearance of the attribute table and how it relates to the map.

  • Options – Provides a dropdown to reveal the table options, which include:
    • Show Selected Records – Displays only the selected records in the table.
    • Show Related Records – Displays records from a separate table that have been related to the attribute table via a common field.
    • Filter – Displays records in the attribute table that meet specific criteria based on expression(s) applied to the table values.
    • Show/Hide Columns – Limits the visibility of columns (fields) in the attribute table.
  • Filter by map extent – Limits the records displayed in the table to only those features that appear in the map. Click the Filter by map extent button to remove this filter and view all records.
  • Selected Community Only/Show All Community Records – Displays the records for only the selected community in the Attribute Table (community layers only).
  • Zoom to – Zooms the map to the extent of the selected features.
  • Clear selection – Clears the table and map selection.
  • Refresh – Refreshes the table.

Resize table

Hover over the divider tab to see a double-arrowed cursor. Pull the tab up to increase the size. Pull the tab down to reduce the size.

Click the down arrow to hide the attribute table.

Show/hide columns

Click the plus button to view a pop-up with a list of the columns (fields) for the data layer. Check the boxes next to the column names to hide or show the column in the table.

Column (field) names

Click a column name to see options for sorting the column by field values and obtaining statistics on the values.
