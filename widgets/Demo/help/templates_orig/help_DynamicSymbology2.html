Tools in the Dynamic Symbology pop-up can be configured to to change the layer's classification scheme, colors and symbols, and transparency.

Classification scheme

By default, most EnviroAtlas layers display with a seven-class geometric interval classification scheme. To change the classification scheme, select one of the classification methods from the Classification dropdown:

  • equal–interval – classes are split into equal value ranges based on the minimum and maximum values.
  • natural–breaks – classes are based on natural groupings in the data, maximizing differences between classes.
  • quantile – classes are created by assigning an equal number of features per class.
  • standard–deviation – classes are separated by standard deviations from the mean value.
  • manual – classes may be manually created.

Next, select the number of classes you want in your custom classification. Up to 20 classes may be created.

Click Apply to make your changes. The classification ramp in the Dynamic Symbology pop-up displays the new ranges and the map display and legend change accordingly.

Colors and symbols

The Symbology button provides several options for polygon fills, outline colors and styles, and point symbol types for point layers.

Select a color ramp from the FILL list, and change the OUTLINE if desired. Click OK, then Apply to make your changes.

Transparency

Adjust the transparency of the layer in the interactive map by using the slider bar.

Reset

To revert to the default classification and symbology, click the Reset button.
