EnviroAtlas data layers are organized in groups, color coded based on the theme of the data.

Click a theme to view a list of data layers associated with that theme.


Within each data layer you will find informational icons and ways to interact with the layer

  • Layer selection - check the box to add the layer to the map.
  • Geographic extent - indicates National- or Community-scale layer.
  • Layer description - read a short description of the layer and access the layer's Data Fact Sheet. Video Example [Exit EPA]
  • Data type - indicates the type or format of the layer as displayed in the map. Click an icon to reveal what they mean.
  • Ecosystem services categories - indicates to which EnviroAtlas ecosystem services benefit categories the layer belongs. A colored icon indicated the layer belongs to the category, otherwise the icon is grayed. Click an icon to reveal what they mean. Video Example [Exit EPA]

Click Expand to reveal all layers in the groups. Click Hide Icons to provide a more compact view of the layers by removing the descriptive icons.

Search for layers using the text box. Type a word or phrase to find specific layers.

Click the filter icon to narrow your selection of groups and layers. Select by data group and/or by geographic extent and only those layers will display in the layer selection panel.
