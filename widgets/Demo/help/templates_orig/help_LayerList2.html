Layers are displayed in the Layer List in the order in which they are added to the map with the exception of Boundaries and Natural Features layers which are added to the bottom of the List. The order of the layers in the Layer List dictate the order the layers draw in the map.

Layer List functions that act on individual layers

  • Caret – Reveal the layer's legend or the sublayers if the item is a layer group.
  • Checkbox – Activate the layer or layer group in the map.
  • Ellipsis – Access a set of tools to configure the layer's appearance in the map or to view layer information.

Caret

Click the caret to expand the layer. For most layers, this will display the layer's legend. For layer groups, clicking the caret reveals the sublayers in that group. Each sublayer has its own caret; click to reveal the layer's legend.

Checkbox

Click the checkbox to activate the layer in the map.

Multiple layers may be displayed simultaneously.

Ellipsis

Click the ellipsis to reveal tools for the layer.

  • Transparency – Adjust the transparency of the layer or the sublayers if the item is a layer group.
  • Draw Layer On Top – Make the layer draw on top of other layers in the map.
  • Change Appearance – Open the Dynamic Symbology tools to change the layer's colors and symbols in the map.
  • Remove Layer – Remove the layer from the map and Layer List.
  • Map Description – View a short description of the layer.
  • Data Fact Sheet – Open the EnviroAtlas Data Fact Sheet PDF in a new browser tab that provides a detailed description of the layer.
  • Access Web Service – Open the ArcGIS Server web service REST page for the layer in a new browser tab.
  • Metadata – Open the layer's metadata record from the EPA metadata repository in a new browser tab. A data download link is available on the metadata page for most layers.
  • Open Attribute Table – Open the attribute table to display the fields in the layer's database.

Layer List functions that act on all layers

  • Legend – View the pop-up displaying the map legend for all active layers in the Layer List.
  • Search – Search the layers in the Layer List by keyword. Layers with the keyword present remain in the List while others are temporarily removed. Click Cancel to return the full list.
  • All Layers Menu – Access a set of tools that act on all layers in the Layer List.
