You may select how to view community data layers in the map. Community layers are represented by a "C" icon in the layer selection panel on the left as well as in the Layer List. Your selection controls how the community data layers are displayed in the map as follows:

  • Combined Communities – The default, displays the legend based on all communities (i.e., comparable across all communities)
  • Individual community – Displays the legend for the selected community's data only.

Select the community interactively in the map or by using the Community Selection tool.

Interactive Map Selection

To interactively select the EnviroAtlas community from the map, first choose a community data layer from the layer selection panel. Circles will show locations in the map; click on the circle for the desired community.

A pop-up appears with a button to Select this community. Click the button to select the community.

Community Selection Tool

To select from the Community Selection tool, click the button next to the community you would like to select, or select Combined Communities.

Selecting an individual community will zoom the map to that community and display the community name in the upper-left corner of the map. Community data layers display based only on the selected community's data.

To select a new community, open the Community Selection tool and choose a different community or select “Combined Communities” to return to viewing the symbology for all communities.
