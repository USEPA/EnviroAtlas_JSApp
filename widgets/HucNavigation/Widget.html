This tool is an initial release (prototype) of the national HUC catchments and is still under review and testing. The Navigate HUC 12 Catchments Tool allows users to click on any location in the conterminous US, navigate to the nearest stream, and navigate upstream or downstream HUCs within the stream network. On the Filter and Search page of this tool, users may choose to select HUCs either by filtering for distance (stream kilometers) or by time of travel (days) from an interactively-selected point. Click on Select Point to begin using this tool. For additional guidance on this tool, please see the About EnviroAtlas document - accessible from the map viewer Help button.
