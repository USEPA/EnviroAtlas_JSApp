Select a location on the map and see the general raindrop flow path and distance to the nearest water feature as depicted in National Hydrography Dataset Plus V2.

This tool is built using nationally available 30 meter elevation data. The data does not account for buildings or engineered stormwater infrastructure such as culverts and pipes and is not intended for use in heavily engineered locations.

Only the last point selected will be displayed in the results.
